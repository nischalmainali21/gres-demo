
## Getting Started
Run Locally
- Clone the repo.
- install the required dependecies running:
```bash
npm install
# or
yarn install
# or
bun install
```
- Start the server:
```bash
npm run dev
# or
yarn dev
# or
bun dev
```
Open http://localhost:3000 with your browser to see the result.
